Early Childhood Education enrolments. Reset All

Data sourced from the Ministry of Education ECE participation statistics in Education Counts, specifically the number of enrolments in early childhood education (.xlsx).

Note: Data collection and definition change

In 2014, the method for data collection changed and around 40% of services completed the Annual Census using the Ministry's new electronic collection tool for ECE: ELI. For these services, the data shown relates to attendances in ECE licensed services, not enrolments. This is a change to the definition of the data and means that the data should not be compared to previous years.

For a more complete description of the data please refer to the source linked above.
